-install virtualenvwrapper - pip install virtualenvwrapper
-create a virtualenv - mkvirtualenv envname
-activate virtualenv - workon envname
-install django in the venv- pip install Django
-create django project - django-admin createproject projectname
-create django app inside the project - python manage.py startapp appname


VIRTUAL ENV :
-see list of virtual envs in the pc - lsvirtualenv
-deactivate virtualenv - deactivate
-remove venv from the pc - rmvirtualenv envname
